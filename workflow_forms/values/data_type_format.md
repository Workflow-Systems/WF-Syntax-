# DataTypeFormat

## Шаблон DataTypeFormat <a href="#template" id="template"></a>

```xml
<DataTypeFormat Type="" Format="" TrueTitle="" FalseTitle="">
</DataTypeFormat>
```

## Атрибуты DataTypeFormat <a href="#attributes" id="attributes"></a>

### Type <a href="#type" id="type"></a>

Название типа данных.

Обязательный атрибут.

### Format <a href="#format" id="format"></a>

Задает формат строки, в которую будет преобразовываться значение.

Необязательный атрибут.

### TrueTitle <a href="#true_title" id="true_title"></a>

Значение, отображаемое вместо значения True, если выбран тип [BooleanDataType](../data_types/boolean_data_type.md).

Необязательный атрибут.

### FalseTitle <a href="#false_title" id="false_title"></a>

Значение, отображаемое вместо значения False, если выбран тип [BooleanDataType](../data_types/boolean_data_type.md).

Необязательный атрибут.
