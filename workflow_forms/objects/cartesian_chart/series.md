# Series

## Шаблон Series <a href="#template" id="template"></a>

```xml
<Series Name="" Color="">
  <Title></Title>
  <Points></Points>
  <Visible></Visible>
</Series>
```

## Атрибуты Series <a href="#attributes" id="attributes"></a>

### Name

Задает имя серии.

Необязательный атрибут. Любое значение будет переведено в текст.

### Color

Задает цвет серии.

Необязательный атрибут. Ожидается имя одного из цветов, описанных на форме или описание цвета в формате HTML (#rrggbb).

## Тэги Series <a href="#specific-tags" id="specific-tags"></a>

### Title

Отображаемое для пользователя наименование серии.

Необязательный тег. Любое значение будет переведено в текст.

По умолчанию используется пустая строка.

```xml
<Title>Имя серии</Title>
```

### Points

Массив данных по точкам.

Необязательный тег. Ожидается матрица со столбцами:

* Значение по вторичной оси (ось X). Ожидается текст или число. Обязательный.
* Значение по основной оси (ось Y). Ожидается только число. Обязательный.
* Текст подсказки при наведении на точку диаграммы. Необязательный. По умолчанию формируется строка из наименования серии, указанного в тэге `<Title>`, и значения по основной оси.

```xml
<Points>
  <DataConnection SourceDataConnection="PrimaryGetDataConnection">
    <Fields>
      <Field Name="MainAxisTextValue" />
      <Field Name="MainAxisValue" />
      <Field Name="Hint" />
    </Fields>
  </DataConnection>
</Points>
```

### Visible

Признак видимости серии.

Необязательный тег. Ожидается логическое значение.

По умолчанию используется значение True.

```xml
<Visible>False</Visible>
```
