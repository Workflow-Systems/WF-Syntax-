# ColorSchema

## Шаблон ColorSchema  <a href="#template" id="template"></a>

```xml
<ColorSchema Value="" >
  <LineColor></LineColor>
  <ThumbFirstColor></ThumbFirstColor>
  <ThumbSecondColor></ThumbSecondColor>
  <ThumbBorderColor></ThumbBorderColor>
  <BarFirstColor></BarFirstColor>
  <BarSecondColor></BarSecondColor>
  <BarThirdColor></BarThirdColor>
  <ElapsedFirstColor></ElapsedFirstColor>
  <ElapsedSecondColor></ElapsedSecondColor>
  <ElapsedThirdColor></ElapsedThirdColor>
</ColorSchema>
```

## Атрибуты ColorSchema <a href="#attributes" id="attributes"></a>

### Value

Задает название цветовой схемы, предопределенной в платформе.

Необязательный атрибут. Ожидается одно из значений: Default, Red, Green или Blue.

По умолчанию используется значение Default.

## Тэги ColorSchema  <a href="#specific-tags" id="specific-tags"></a>

### LineColor <a href="#line_color" id="line_color"></a>

Цвет делений на шкале.

Необязательный тэг. Ожидается имя одного из цветов, описанных на форме или описание цвета в формате HTML (#rrggbb).

По умолчанию используется значение, определенное выбранной цветовой схемой.

```xml
<LineColor>LineColorName</LineColor>
```
