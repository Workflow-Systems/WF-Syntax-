# ContextMenu

## Описание ContextMenu <a href="#description_context_menu" id="description_context_menu"></a>

В значении тэга `<ContextMenu>` ожидается список тэгов [`<MenuItem>`](../menu_item/).

```xml
<ContextMenu Name="" Type="">
  <!--Тэги, общие для всех контекстных меню-->
  <!--Тэги, специфичные для определенного контекстного меню (зависит от типа)-->
</ContextMenu>
```

Контекстные меню значения не имеют.
